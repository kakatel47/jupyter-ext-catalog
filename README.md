# Jupyter Extensions Catalog

Веб-каталог расширений для Jupyter Notebook на чистом HTML/CSS/JS.
Позволяет просматривать расширения, настраивать их (например, текст ячейки по умолчанию)
и скачивать готовый ZIP-архив для установки в Jupyter. Избранное сохраняется в localStorage.

## 🛠 Стек
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Библиотеки:** JSZip (генерация ZIP на клиенте)
- **Хранение:** localStorage (избранное и настройки)
- **Deploy:** GitHub Pages

## 🚀 Локальный запуск
Просто открой `index.html` в браузере или через Live Server в VS Code.
